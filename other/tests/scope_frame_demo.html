Important Points

  1. When creating frame images, each should extend fully to cover its corners.
  2. The corners appear in both frames they belong to (e.g., upper-left corner appears in both UPPER and LEFT frames).
  3. Make sure the design of the corner in one frame matches the same corner in the other frame.
  4. Frame images should have transparent areas where the diorama content shows through.
  5. Opaque areas should match the page background color (#202029).

Example PNG Design

For a frame_upper_1_202029.png:

  • Create a PNG with width matching your diorama width (e.g., 600px)
  • Height sufficient for your frame design (e.g., 100px)
  • Fill with solid #202029 color
  • Create a transparent cutout in the shape you want (like a curved, jagged, or twig-like bottom edge)
  • Ensure the cutout extends to the full width, including corners
  • The transparent area is where the diorama content will show through
