Scope Frames Test

This page demonstrates how scope frames can be used to create non-rectangular diorama windows.

The example below shows placeholder frames using CSS clip-paths to simulate the effect until you create your own frame PNG assets.

To implement with real images, place your transparent PNG frame assets in the appropriate folders and configure them in your scope.json file.

Implementation Guide

To implement scope frames in your dioramas:

{
  "frames": {
    "upper": "jagged_edge.png",
    "lower": "rough_border.png",
    "left": "torn_edge.png",
    "right": "torn_edge.png"
  },
  "filter": "subtle_grain.png"
}

See the full documentation in the SCOPE_FRAMES_GUIDE.md and CREATING_SCOPE_FRAME_ASSETS.md files.
